# Zombiesix's Skills

一个精选的 [Agent Skills](https://agentskills.io/home) 集合，反映 [Zombiesix](https://github.com/Zombiesix) 的偏好、经验和最佳实践，以及工具的使用文档。

> [!NOTE]
> 这是一个前端开发相关的 Skills 集合项目，目前包含 Vue3 相关的 Skill，后续会逐步添加更多前端技术栈的 Skills。欢迎反馈和贡献。

## 安装

```bash
pnpx skills add Zombiesix/skills --skill='*'
```

或全局安装所有 Skills：

```bash
pnpx skills add Zombiesix/skills --skill='*' -g
```

了解更多 CLI 用法请访问 [skills](https://github.com/Zombiesix/skills)。

## Skills

此集合专注于前端开发，提供项目初始化和最佳实践相关的 Skills。

### 手工维护的 Skills

> 观点鲜明

由 Zombiesix 手工维护，包含偏好的工具、设置惯例和最佳实践。

| Skill | 描述 |
|-------|------|
| [vue3-template](skills/vue3-template) | 完整的 Vue3 + TypeScript 项目初始化，包含 ESLint、Prettier、Vue Router、Pinia、VueUse、Naive UI、Axios 等全套配置 |
